Basic Example

The simplest integration. A chat widget appears in the bottom-right corner.

Features

Auto theme Proxy-only auth Bottom-right position Welcome message

Integration Code

<script src="https://cdn.example.com/goclaw-webchat.umd.js"></script>
<script>
  GoClaw.init({
    url: 'wss://proxy.example.com/ws',
    title: 'Chat with us',
    theme: 'auto',
  });
</script>
ⓘ The url points to your proxy server. The gateway auth token is kept server-side and never exposed to the browser.
