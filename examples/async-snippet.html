Async Snippet Non-blocking

Non-blocking script loader pattern. Identical to how Intercom, Drift, and similar widgets load.

Features

Async loading Queue-based init Non-blocking Copy-paste ready

Copy-Paste Snippet

<script>
  (function(w,d,s,u){
    w.__goclaw_queue=w.__goclaw_queue||[];
    w.GoClaw=w.GoClaw||{
      init:function(){
        w.__goclaw_queue.push(['init',arguments[0]])
      }
    };
    var f=d.getElementsByTagName(s)[0],
        j=d.createElement(s);
    j.async=true;
    j.src=u;
    f.parentNode.insertBefore(j,f);
  })(window,document,'script',
    'https://cdn.example.com/goclaw-webchat.umd.js');

  GoClaw.init({
    url: 'wss://proxy.example.com/ws',
    title: 'Chat with us',
    theme: 'auto',
  });
</script>
ⓘ Commands called before the script loads are queued and replayed automatically. The page never blocks on the widget script.
